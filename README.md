# Travian_oasis_calculator
Travian calculator to maximize oasis farms (by time, troop count , resources lost )  
Usage:  
run ofinder.py with your jwt key to recieve a txt file with close by oasis  
run app.py   
open index.html on browser (just a simple ui for selecting what to attack)  
add requested values of troops levels etc...  
upload text file from ofinder  
press "analisar oasis"   
farm a lot
